浏览器内核
浏览器内核主要分为渲染引擎（layout engine和 Rendering engine，负责网页内容以及css渲染）和js引擎（解析和执行js来实现网页的动态效果）。1）Trident内核：IE,360等；2）Gecko：ff等；3）Presto：Opera；4）webkit:Safari,chrome等
H5离线储存
含义：在没有连接网络时，也可以正常访问站点或者应用，在联网后，更新机器上设置的那些缓存文件。
原理：HTML5的离线存储是基于一个新建的.appcache文件的缓存机制(不是存储技术)，通过这个文件上的解析清单离线存储资源，这些资源就会像cookie一样被存储了下来。之后当网络在处于离线状态下时，浏览器会通过被离线存储的数据进行页面展示。
使用： 1、页面头部像下面一样加入一个manifest的属性； 2、在cache.manifest文件的编写离线存储的资源； CACHE MANIFEST #v0.11 CACHE: js/app.js css/style.css NETWORK: resourse/logo.png FALLBACK: / /offline.html 3、在离线状态时，操作window.applicationCache进行需求实现。
iframe缺点
1、iframe会阻塞主页面的onload事件 2、搜索引擎无法解读 3、iframe和主页面共享连接池，而浏览器对相同域的连接数有限制，影响页面并行加载

1、原因是chrome默认给自动填充的input表单赋予黄色背景，代码为 
	
						
							input:-webkit-autofill{
								background-color:#faffbd;
								background-image:none;
								color:#000;
							}

2、改变的方法（主要是覆盖浏览器赋予的颜色）： 1）纯色背景（如果有圆角，也可以增加在下列代码中） 
	
						
							input:-webkit-autofill{
								background-color:#fff;
								background-image:none;
								color:#000;
							}
2）有图片背景 1、图片简单，则使用1）中的内阴影来实现； 2、复杂只能牺牲chrome自动填充表单的功能，使用 js 去实现
